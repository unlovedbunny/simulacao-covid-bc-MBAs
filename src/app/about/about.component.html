General Information

Predictions made with this tool hold no real world relevance. This tool is supposed to visualize the risc of pandemic spread without mitigation - it is not supposed to inform your decisions about health and safety. For detailed information on ongoing pandemics, please consult relevant sources such as the CDC or the WHO.

The traffic model

SIR network simulations require a network model, i.e. a description of how individual regions interact. In this simulation, we use flight data to couple the countries. Since I was not able to find individual data for every country, I did this as follows: for every country I know how many aircrafts enter and leave. Based on this, I can compute which fraction of all global flights land or start in a given country. So if I need to know how many people travel from A to B, I simply multiply the total daily air-travellers with the fractions of planes leaving A and the fraction of planes arriving in B. While this is not perfect, it offers an easy implementation and can be refinded later.
